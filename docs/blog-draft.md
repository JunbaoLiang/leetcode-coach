# 我为什么给自己造了一个"不帮我解题"的刷题 AI

> 技术博客草稿 · 2026-07 · 配合 [LeetCode Coach](https://github.com/JunbaoLiang/leetcode-coach) 开源发布

## 起点:一个转行者的真实痛点

我是计算化学 PhD,正在转向 MLE / AI4S 方向。刷题时我发现自己的问题从来不是"这道题解不出来"——把题丢给任何一个 LLM,几秒钟就有漂亮的题解。问题是:**三周后我自己还是不会做**。AI 把"解出这道题"变得前所未有地便宜,也把"假装自己学会了"变得前所未有地容易。

于是这个项目的核心主张反直觉但简单:AI 的作用不是帮你更快地解题,而是帮你**记住更多做过的题、修复真实的弱点**。

## 调研:现有开源项目缺了什么

动手前我调研了一圈开源的 AI 刷题工具,大致三类:

- **提示器类**(如 LitCoach 形态的插件):在 LeetCode 页面里给提示,但提示不分级、不记录,给完就完,和学习闭环无关;
- **教练类**(如 guanyipengai/leetcode-coach):有"提示→归档→讲解"的雏形,给了我教练闭环的启发,但没有复习调度;
- **调度类**(如 eannchen/leetsolv):做了 SM-2 间隔复习,但质量分完全靠自评——而自评恰恰是最不可信的信号(刚看完题解的人总觉得自己"想起来了")。

两个空白几乎没人做:**ML coding 赛道**(numpy 手写 k-means、反向传播、attention——MLE 面试的硬通货,现有工具全部只覆盖 SWE 算法题)和**多人群适配**(应届、转行、MLE 的课程配比和面试 bar 本就不同)。

## 四个核心设计决策

### 1. 证据驱动的 SM-2:自评只是输入之一

经典 SM-2 的质量分 q 靠用户自评。我把它改成合成信号:

```
q = 自评回忆度 − 0.5×最深提示等级 − 0.5×min(判题失败次数, 3) + 一次AC加成
```

你说你"毫不费力"?但你要了 L3 提示、判题挂了两次——系统看到的是 5 − 1.5 − 1 = 2.5,这题明天就会回来找你。调度器是纯函数,17 个单测钉死每条公式,重要题(importance=4)的复习间隔还会缩短 25%。

### 2. 刻意摩擦:求助的不便是特性

提示必须逐级手动请求(L1 方向性提问 → L2 关键观察 → L3 算法骨架 → L4 投降),每次只升一级,教练在当前级别只会"换角度再启发"。没有"一键看答案"按钮;记录表单不允许跳过自评。目的只有一个:**保留挣扎时间,那是学习真正发生的地方**。这条原则写进了 CONTRIBUTING——任何降低摩擦的 PR 需要先书面论证。

### 3. 两个人格,严格分离

练习模式的教练随叫随到但绝不剧透;模拟面试的面试官全程不出戏——不提示、不安慰、不确认对错,你沉默太久它只会说"请继续说出你的思考"。结束后才切换出人格,产出六维 rubric、引用具体对话轮次的复盘("我两次追问 worst case 你才给出,见第 4、11 轮")和补练处方。实测中这种"不给糖"的设计比任何鼓励式 AI 都更接近真实面试的压力感。

工程上,两个人格是两个 markdown prompt 文件。所有 prompt 都以文件形式存在、所有 LLM 评估输出都是过 Pydantic 校验的结构化 JSON 落库——prompt 可以像代码一样 diff、review、回滚。

### 4. 版权红线与浏览器内判题

算法题只存元数据(题号/标题/难度/pattern),题面和判题永远发生在 LeetCode 官网——190 条元数据逐条通过官方 GraphQL 核对(顺便发现我记忆中的官方入门题单和真实题单几乎完全不同,LLM 的记忆不可信,验证管线才可信)。

ML 赛道的 10 道题是原创的,判题走了一条取巧的路:**Pyodide 在浏览器里跑 numpy**。服务端执行用户代码的沙箱问题直接消失,判题延迟为零,失败信息只报形状与容差("返回形状 (3,) 期望 (4,)"),不泄露期望值。代价是测试规格对 devtools 可见——对一个学习工具而言,想作弊的人骗的只是自己。

## 一些工程细节

- **全链路可回滚**:CI(ruff + pytest + Alembic 迁移往返 + 前端构建)保证 main 每个提交可运行;每个迁移必须带可执行的 `downgrade()`;用户数据有 JSON 快照级的 `make backup/restore`。
- **渐进增强的鉴权**:本地 dev 是零配置单用户;生产环境一个 `AUTH_ENABLED=true` 切换到 GitHub OAuth 多用户,数据按用户隔离,越权访问一律 404(不确认资源存在)。
- **成本**:全部 LLM 调用走 claude-sonnet-5,重度使用一个月 2-3 美元——比一杯奶茶便宜的私人教练。
